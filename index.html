Linux-Server-Configuration

Take a baseline installation of a Linux distribution on a virtual machine and prepare it to host web applications, to include installing updates, securing it from a number of attack vectors and installing/configuring web and database servers. This project is linked to Udacity's Configuring Linux Web Servers course.
